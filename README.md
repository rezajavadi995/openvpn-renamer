# 🌐 OpenVPN Renamer

A powerful Termux tool to rename `.ovpn` files with smart suggestions, usage tracking, and clean OpenVPN-compatible metadata editing.

---

## ✨ Features

- 🔁 Rename `.ovpn` configuration files safely  
- 🧠 Smart suggestion system (remembers frequently used paths)  
- 📊 Tracks file usage (3+ times → appears in suggestions)  
- 🎯 Interactive CLI menu  
- 🎨 Colored terminal UI (Termux optimized)  
- 🧾 Safe OpenVPN-compatible editing  
- 🏷️ Injects custom profile name inside config  
- 📁 Keeps files in original directory
